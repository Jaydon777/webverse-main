# webverse
# Web App

This is a web application built using Express.js, EJS, and other dependencies. It provides functionality for student-related operations, including login, signup, and accessing different pages.

## Installation

1. Clone the repository:

   ```bash
   git clone https://github.com/Aadityaa2606/webverse.git
2. Install the dependencies:
   ```bash
   npm install

3. Start the server:
   ```bash
   node app.js
   ```

# Routes implemented
# STUDENT
- **Home Page:** `/`
- **Student Dashboard:** `/student`
- **Student Authentication Page:** `/student/auth`
- **Login Form Submission:** `/handleLogin`
- **Signup Form Submission:** `/handleSignup`
- **Student Leave Page:** `/student/leave`
- **Student Complaint Page:** `/student/complaint`
- **Student Course Page:** `/student/course`
- **Student Event Page:** `/student/event`
- **Student Mess Page:** `/student/mess`
- **Student Room Page:** `/student/room`
